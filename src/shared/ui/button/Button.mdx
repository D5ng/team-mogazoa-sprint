# Button

버튼 컴포넌트는 총 3가지의 타입을 받아 스타일들을 적용할 수 있습니다.

`isLoading` 및 `disabled`를 사용해 사용자한테 로딩중인지, 버튼의 클릭이 활성화 되어있는지 인지할 수 있도록 사용해주는것이 좋습니다.

- Primary
- Secondary
- Tertiary

유연한 사용성을 위해 `className`을 받아서 원하는대로 적용할 수 있습니다.

❗️프로젝트 특성상 버튼의 사이즈가 부모 태그의 너비를 기반으로 하기 때문에 사이즈 속성은 없습니다.
