# Dropdown

드롭다운 컴포넌트는 2가지의 스타일을 사용할 수 있습니다.

- **`border가 있는 드롭다운`**
- **`border가 없는 드롭다운`**

드롭다운 컴포넌트는 `className`을 통해 유연한 스타일을 적용할 수 있습니다. 드롭다운의 타이틀은 항목의 기본값으로도 설정할 수 있으며, 플레이스홀더로도 사용할 수 있습니다.

**❗️드롭다운의 기본 스타일은 `border`가 없는 스타일입니다.**
