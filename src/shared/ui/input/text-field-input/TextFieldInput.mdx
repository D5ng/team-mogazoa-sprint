# TextFieldInput

`TextFieldInput` 컴포넌트는 사용자로부터 텍스트 입력을 받는 데 사용됩니다. React Hook Form과 통합되어 있어 폼 관리와 validation을 쉽게 할 수 있습니다.

## 주요 기능

- **placeholder**: 사용자에게 입력 지침을 제공합니다.
- **validation**: React Hook Form과 연동하여 입력값의 유효성을 검사합니다.
- **error**: Validation 실패 시 error 메시지를 표시합니다.
- **setValue**: `setValue` 함수를 통해 필드 값을 프로그래매틱하게 설정할 수 있습니다.
