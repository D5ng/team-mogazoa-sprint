# TextareaInput

`TextareaInput` 컴포넌트는 여러 줄의 텍스트 입력을 받는 데 사용됩니다. React Hook Form과 통합되어 있어 폼 관리와 validation을 쉽게 할 수 있습니다.

## 주요 기능

- **placeholder**: 사용자에게 입력 지침을 제공합니다.
- **maxLength**: 입력 가능한 최대 문자 수를 설정합니다.
- **validation**: React Hook Form과 연동하여 입력값의 유효성을 검사합니다.
- **error display**: Validation 실패 시 error 메시지를 표시합니다.

**❗️프로젝트 특성상 textarea의 사이즈가 부모 태그의 너비를 기반으로 하기 때문에 사이즈 속성은 없습니다.**
