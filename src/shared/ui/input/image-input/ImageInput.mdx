# ImageInput

`ImageInput` 컴포넌트는 사용자가 이미지를 업로드하고 미리보기를 할 수 있도록 도와줍니다. 이미지 업로드 취소, 미리보기, 업데이트 여부 표시 등 다양한 옵션을 제공합니다.

## 주요 기능

- **`previewImage`**: 업로드 전에 미리보기할 이미지의 URL을 설정할 수 있습니다.
- **`isUpdated`**: 이미지가 업데이트되었는지 여부를 나타냅니다.
- **`onSuccess`** 및 **`onCancel`**: 업로드 성공 및 취소 작업을 처리하는 핸들러를 제공합니다.
