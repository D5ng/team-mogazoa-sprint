# CategoryDropdown

카테고리 드롭다운은 내부적으로 상태를 관리하고 있는 제어 컴포넌트 입니다. React Hook Form과 사용하기 위해서 useController를 내부적으로 사용하고 있습니다. 따라서 React Hook Form을 사용할 때에는 control과 errors 객체를 넘겨주어 사용해야 합니다.
